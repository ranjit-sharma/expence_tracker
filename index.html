📊 Personal Expense Tracker

Language: C++

🔹 Description

This is a simple console-based application that allows users to add, view, and manage daily expenses. It's written in C++ and stores data in a text file.

🔹 Features

  • Add daily expense records (amount, category)
  • Display all recorded expenses
  • Filter expenses by category
  • Save data in a file

🔹 Sample Code

#include <iostream>
#include <fstream>
using namespace std;

int main() {
    ofstream file("expenses.txt", ios::app);
    string category;
    float amount;
    cout << "Enter category: ";
    cin >> category;
    cout << "Enter amount: ";
    cin >> amount;
    file << category << " " << amount << endl;
    file.close();
    cout << "Expense added!" << endl;
    return 0;
}

🔹 How to Install & Run

  • ✅ Make sure a C++ compiler like g++ is installed.
  • ✅ Save the code in a file named expense_tracker.cpp.
  • ✅ Open terminal / command prompt in the file's directory.
  • 🔧 Compile using: g++ expense_tracker.cpp -o tracker
  • 🚀 Run using: ./tracker (Linux/macOS) or tracker.exe (Windows)
